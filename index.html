About Me

Colin Boyette is a musician, artist, and software engineer based in Charlotte, North Carolina. He is dedicated to building high quality software and video game assets with an emphasis on elegance and design.

Projects

Untitled Adventure Game

Game development project recently ported to Godot 4. Features all original assets and code, including a custom dialogue system, custom entity pathfinding behavior, and unique shaders for edge detection, point mesh scaling, and procedural clouds.

'Adrift' Fall 2022 Game Jam Submission

Submission for the UNCC Game Development Club's 49 Hour Game Jam. Features all original assets, including an original song. Steer your leaf close to the tree to gain points, but don't crash!

Multiplayer Space Shooter

Prototype research project to better understand Godot 4's new networking nodes. Jetpack around a small solar system with your friends. Supports peer-to-peer LAN multiplayer.

3D Level Builder for GMS 2.0

Research project in GameMaker Studio 2.0 that allows for building and rendering simple 3-dimensional environments using an original library of scripts.

Pixel-Perfect Vector-Based Collision Handling for GMS 2.0

Research project in GameMaker Studio 2.0 that intends to solve common collision handling problems in pixel-based projects. Uses vector math for high reliability.
